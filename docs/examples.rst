Loop and Conditional
--------------------

   Bhaiyon aur Beheno

   Vikas hai n: int ka 3

   Agar desh == "x" toh:
       Mitron! boliye "sahi"
   Nahi toh:
       Mitron! boliye "uh oh ho gaya"

   Desh ke liye N baar(n):
       Mitron! boliye "jeff"
